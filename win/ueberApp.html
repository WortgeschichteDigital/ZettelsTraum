Changelog

Version 0.2.09. April 2019

Verbesserungen

  • Suchfunktion: Das Finden von Wörtern mit Sonderzeichen (ẽ und ñ) wurde verbessert.
  • Belegtext: Beim Kopieren wird die Quellenangabe automatisch mitkopiert. Dies betrifft alle Kopierfunktionen: das Kopieren via Icon und das Kopieren einer Textauswahl via Kontextmenü; beides sowohl in der Belegliste als auch im Belegzettel.
  • Belegzettel: Die Texttools erkennen illegales Nesting jetzt in mehr Fällen als zuvor.

Behobene Fehler

  • Belegzettel: Das Einfügen von Text im Belegfeld durch Klick auf das Einfüge-Icon funktionierte nicht.
  • Belegzettel: Beim Einfügen von Text im Belegfeld durch Klick auf das Einfüge-Icon wurden einfache Zeilenumbrüche im HTML-Quelltext ignoriert.
  • Belegzettel: Ein Hinweis auf illegales Nesting erschien nicht in jedem Fall.
